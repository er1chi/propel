# Example Integrations (NestJS)

NestJS-related example repositories from [MikroORM Example Integrations](https://mikro-orm.io/docs/examples).

---

## NestJS Examples

- [RealWorld example app (Nest + MySQL)](https://github.com/mikro-orm/nestjs-realworld-example-app) — Full RealWorld spec with NestJS and MikroORM
- [NestJS + MySQL + TypeScript](https://github.com/mikro-orm/nestjs-example-app) — Basic NestJS integration starter
- [Nest + Shopify + PostgreSQL + GraphQL](https://github.com/Cloudshelf/Shopify_CSConnector) — Production NestJS app with PostgreSQL and GraphQL

> Note: Some examples use MySQL but demonstrate NestJS patterns (`MikroOrmModule`, `@InjectRepository`, request context) applicable to PostgreSQL projects. Adapt driver package to `@mikro-orm/postgresql`.
